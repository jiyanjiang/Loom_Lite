# Loom_Lite

Science for everyone in the age of AI.

Loom_Lite is a lightweight AI-powered scientific workflow tool built with Gradio.

## 🚀 Features (v0)

- AI-assisted scientific exploration workflow
- User-controlled API key usage
- Modular workflow structure
- Local-first execution (Windows / Mac compatible)

## 🧪 Goal

Loom_Lite aims to help users:

- Explore interdisciplinary science
- Build structured knowledge maps
- Transform reading into research workflows

## 🛠 Tech Stack

- Python 3.11+
- Gradio
- GitHub CLI / Git
- Cloudflare (future deployment)

## 📦 Status

v0 initial release (prototype)

## 🔜 Next Steps

- Workflow modularization
- Knowledge graph integration
- Cloud deployment
- Subscription-based advanced workflows
